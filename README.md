# RoadRulesSim
Проект для визуализации и моделирования дорожных ситуаций

Проект для визуализации и моделирования дорожных ситуаций. Процедурно генерируется город, со знаками ПДД и светофорами. Боты ИИ в виде машин перемещаются по городу соблюдая правила. Игрок управляя автомобилем должен проехать от точки А до точки Б соблюлдая ПДД.
